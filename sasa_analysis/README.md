# ./SASA.py #

# Overview #

This script computes solvent-accessible surface area (SASA) from MD trajectories using a custom implementation of the Shrake–Rupley algorithm with MDAnalysis. It:

 - Samples atom surfaces using a spherical point grid
 - Uses a KD-tree for efficient neighbor occlusion checks
 - Supports atom selection (e.g. aromatic groups)
 - Computes SASA over trajectory frames and basic statistics

# Usage #

`python script.py <input_dir> <structure.pdb>`

Arguments

`<input_dir> — directory containing *.xtc trajectories`

`<structure.pdb> — reference, NAMED_structure. Atom selection relies on the resids and atom names generated by urethane_top_MD`

# Output #

sasa_aromatic.csv → SASA values per frame

Additionally printed:

 - Mean, standard deviation
 - Minimum and maximum SASA with frame indices

Units: Å²

# ./sasa_plot.py #

# Overview #

This script analyzes solvent-accessible surface area (SASA) statistics from MD simulations using outputs generated by MDAnalysis-based workflows. It:

 - Loads per-frame SASA time series (sasa_<type>.csv)
 - Computes mean and standard deviation per system
 - Groups systems by stereochemistry and solvent (CHCl₃ / ACN)
 - Produces comparative summary plots

# Usage #

`python ./sasa_plot.py <sasa_type>`

Argument

`<sasa_type> — suffix of SASA file (e.g. aromatic, total)`

Expected input files:

```bash
analysis/boc-*/sasa_<sasa_type>.csv
analysis-acn/boc-*/sasa_<sasa_type>.csv
```

# Output #

1. Statistics file
`sasa-<sasa_type>.dat`

Contains:

system_name    mean_sasa    std_sasa

2. Comparative bar plot

 - SASA_<sasa_type>_bar.png
 - X-axis: stereochemical classes (SSSS, SSSR, SRSS, SRSR)
 - Bars: mean SASA
 - Error bars: standard deviation

Two solvents:
 - CHCl₃
 - ACN
