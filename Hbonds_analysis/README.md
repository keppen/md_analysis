# ./hydrogen_bonds_matrix.py #

# Overview # 

This script computes hydrogen bond statistics from molecular dynamics trajectories using MDAnalysis. It:

Identifies donor–acceptor pairs based on atom selections
Runs hydrogen bond analysis over trajectory frames
Aggregates results into interaction matrices
Supports parallel processing over trajectory chunks

# Usage #

`python ./hydrogen_bonds_matrix.py <input_dir> <structure.pdb> <topology.itp>`

Arguments
`<input_dir> — directory containing isolated urethane trajectory file/s (e.g. analysis/boc-pgs4/iso/ , ./iso.xtc)`
`<structure.pdb> — reference, NAMED_structure. Atom selection relies on the resids and atom names generated by urethane_top_MD`
`<topology.itp/top> — isolated urethane topology file (for charges)`

# Output #

Saved as NumPy arrays (.npy) with prefix test_hbonds:

test_hbonds_avg.npy → summed H-bond count matrix
test_hbonds_all_matrix.npy → per-chunk matrices
test_hbonds_all_array.npy → raw H-bond events
test_hbonds_indexes.npy → donor/acceptor labels

Matrix shape:

rows → donors

columns → acceptors

# ./HB-solvent-solute.py #

# Overview #

This script computes hydrogen bonding between solute and solvent using MDAnalysis. It:

Automatically detects donors/acceptors for solute (UNK) and solvent (LIG)
Filters solvent interactions within a distance cutoff
Runs parallel hydrogen bond analysis over trajectory chunks
Builds interaction matrices and separates solute–solute vs solvent interactions (Not enabled)

# Usage #
`python3 ./HB-solvent-solute.py <input_dir> <structure.pdb> <geometry.tpr>`
Arguments

```bash
<input_dir> — directory containing solvated urethane trajectory file 'full.xtc (e.g.  '.' )
`<structure.pdb> — reference, NAMED_structure. Atom selection relies on the resids and atom names generated by urethane_top_MD`
<geometry.tpr> — binary topology file of solvated urethane
```


# Output #

Printed and computed in memory:

Hydrogen bond event array (stacked over all frames)
hb_matrix → solute–solute interaction counts (donor × acceptor)
hb_solvent_donors_array → solvent interactions per donor
hb_solvent_acceptors_array → solvent interactions per acceptor
Sorted residue labels for matrix indexing

(No files are saved unless saving is re-enabled)

# ./hydrogen_bond_matrix_ploting.py # 

# Overview #

This script analyzes hydrogen bond matrices (previously computed with ./hydrogen_bonds_matrix.py) and:

Generates heatmaps of H-bond interaction matrices per system
Sorts donor/acceptor labels for consistent visualization
Compares systems using cosine similarity of flattened matrices

It supports multiple environments (e.g. CHCl₃, ACN) and applies custom naming.

# Usage #
`python ./hydrogen_bond_matrix_ploting.py`

Run in a directory containing:

```bash
analysis/boc-*/hbonds_avg.npy
analysis/boc-*/hbonds_indexes.npy
analysis-acn/boc-*/hbonds_avg.npy
analysis-acn/boc-*/hbonds_indexes.npy
```


# Output #

Per-system heatmaps:

`<System>-Average-Hydrogen-Bond-hbond-matrix.png`

Rows: donors

Columns: acceptors

(Optional, commented):
Cosine similarity heatmaps between systems

Subset comparisons based on stereochemistry or residue type
